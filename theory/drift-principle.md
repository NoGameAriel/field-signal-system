# The Drift Principle

Drift is the baseline condition of all modeled systems under changing reality. No domain remains perfectly aligned over time.

The critical distinction is whether the system retains awareness that drift is occurring.

- Unintentional drift → failure mode (invisible misalignment accumulates)
- Intentional drift → evolution mode (model is consciously updated)

A system does not fail because it changes. It fails when it cannot distinguish change from error.
